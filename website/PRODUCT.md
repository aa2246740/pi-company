# Product

## Register

brand

## Users

**主要用户**: Pi 用户、Codex 用户、开源 agent 工具构建者、开发者和技术团队负责人
- 对 AI 代理协作感兴趣的技术人员
- 希望了解和使用 pi-company 的开发者
- 需要本地多代理协作方案的团队
- 已经会为了一个项目打开多个 Pi/terminal 窗口的人

**用户上下文**:
- 在本地开发环境中工作
- 使用 macOS/Linux/Windows 终端
- 熟悉命令行工具和开发工作流
- 想提高 agent 并行速度，但不愿意把工作交给不可见的后台黑盒

**用户目标**:
- 5 秒内理解 pi-company 是做什么的，为什么值得安装
- 理解 pi-company 的核心概念和价值
- 学习如何安装、配置和使用 pi-company
- 掌握完整的代理协作工作流
- 通过交互式教程获得实践经验

## Product Purpose

pi-company 网站是开源项目首页、官方文档入口和产品体验平台。第一屏必须先完成转化：让陌生人理解“让多个 Pi 像一个可见的本地项目团队一样工作”，再进入功能细节和交互教程。

**核心目标**:
1. **定位**: 用一句话讲清 pi-company 的用途和差异点
2. **转化**: 先回答为什么要装，而不是先堆功能清单
3. **教育**: 用递进结构教会用户如何使用 pi-company
4. **展示**: 通过真实工作流和交互教程展示协作方式
5. **验证**: 作为 pi-company 能力的真实验证项目

**成功标准**:
- 首屏能让陌生开发者理解它是否适合自己
- README 和官网前部先展示最强卖点，再解释实现细节
- 用户能够从零开始学习 pi-company 的完整工作流
- 交互式教程真实可用，而非装饰性占位符
- 所有主要功能都有详细的教程或指南
- 网站能够本地运行并通过生产构建

## Brand Personality

**三个关键词**: 透明、可控、可信

**语调特点**:
- **专业**: 技术准确、信息密集、不废话
- **透明**: 用具体流程、状态和证据说服用户
- **可控**: 强调用户对代理的可见性和可操作性
- **克制**: 终端美学服务于理解，不为炫技牺牲叙事

**情感目标**:
- 信任感: 展示 pi-company 的可靠性和透明度
- 掌控感: 用户能够完全控制代理行为
- 效率感: 展示代理协作带来的效率提升
- 怀旧感: 复古终端风格带来的技术怀旧

**声音特征**:
- 不使用营销术语（如"赋能"、"超级充电"、"无缝"）
- 不使用夸张的承诺
- 使用具体的技术术语和命令
- 保持简洁、直接、技术准确

## Anti-references

**明确避免的设计**:
1. **通用 SaaS 着陆页**: 避免"英雄区域 + 特性网格 + 定价表"的模板
2. **单一绿色调色板**: 不要只用绿色，要使用完整的 CRT 荧光色系
3. **装饰性元素**: 避免装饰性斑点/球体、通用渐变、玻璃态效果
4. **营销废话**: 避免"赋能"、"超级充电"、"无缝体验"等空洞词汇
5. **只堆酷炫 TUI**: 复古终端风格不能替代清晰定位和内容递进
6. **过度动画**: 避免无意义的动画，保持专业感

**参考网站特征** (避免的):
- Stripe/Linear/Vercel 的营销着陆页风格
- 纯文档站点（如 VuePress 默认主题）
- 传统企业软件界面
- 现代 SaaS 仪表板

## Design Principles

1. **实践所宣讲**: 网站本身应该展示 pi-company 的能力，使用 pi-company 的真实概念和术语

2. **展示而非讲述**: 通过交互式模拟器和教程展示功能，而不仅仅是文字描述

3. **专家信心**: 界面应该传达技术专业性，让用户相信这是一个成熟、可靠的工具

4. **递进优先**: 先讲一句话定位，再讲为什么装，再讲怎么用，最后讲完整功能

5. **真实透明**: 不隐藏复杂性，展示真实的工作流程，包括错误和边界情况

6. **渐进式披露**: 从简单概念开始，逐步引入高级功能，但不要过度简化

## Accessibility & Inclusion

**WCAG 级别**: AA 级标准

**已知用户需求**:
- 键盘导航: 所有交互组件必须支持键盘操作
- 屏幕阅读器: 使用语义化 HTML 和 ARIA 属性
- 颜色对比: 确保文本对比度符合 WCAG AA 标准（4.5:1 对比度）
- 动画控制: 提供减少动画的选项（`prefers-reduced-motion`）
- 字体大小: 支持用户自定义字体大小
- 焦点指示: 清晰的焦点状态指示

**特殊考虑**:
- CRT 效果可能对某些用户造成不适，提供关闭选项
- 高对比度模式增强可读性
- 确保在明亮环境下也能正常阅读
- 支持暗色和高对比度主题

**国际化**:
- 当前主要支持英文（pi-company 是英文产品）
- 预留国际化扩展能力
- 使用清晰、简单的英文，避免俚语和复杂句式
