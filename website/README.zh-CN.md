# pi-company 官方网站

[English](README.md) | [中文](README.zh-CN.md)

这是 pi-company 的官方文档和教程网站。默认界面为英文，右上角可切换中文。

## 本地开发

```bash
npm install
npm run dev
```

## 检查

```bash
npm run check
```

`check` 会执行隐私扫描、测试、构建，并在构建后再次扫描。
